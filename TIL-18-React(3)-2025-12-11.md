### 화살표 함수

- 변수 = function() { return }  (함수표현식)
- 변수 = (인자) ⇒ { return {인자} }

```jsx
<button onClick={handleClick}>버튼</button>
```

> handleClick 이라는 변수(함수위치)만 넘겨주고,
> 

> 리액트가 가지고 있다가 클릭되는 순간 알아서 ()를 붙여 실행
> 

```jsx
<button onClick={handleClick()}>버튼</button>
```

> 괄호()를 붙여버리면, 클릭하기도 전에, 페이지가 렌더링 될 때
> 

> 함수가 바로 실행되어 버림
>
