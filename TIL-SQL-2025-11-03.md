## 데이터베이스

- 기본키 : Primary KEY (pk)
- 참조키 : Foreign KEY (fk)

```SQL
SELECT 
  @@@
FROM 
  @@@
ORDER BY
  @@@ DESC

SELECT
  @@@
FROM
  @@@
WHERE
  'country' IN ('Canda','Germany');

SELECT 
  LastName, FirstName
FROM
  customers
WHERE
  "LastName" LIKE '%son';
```

- 와일드카드 연산자
  - '%' : 0개 이상의 문자열과 일치하는지 확인
  - '_' : 단ㅇ릴 문자와 일치하는지 확인
  - _son은 ㅂ나드시 son 앞에 한 문자가 있어야 함
  - LIMIT 7 -> 상위 7개만 조회
  - LIMIT 3,4 (오프셋3 -> 4개) (4,5,6,7 조회)

WHERE - 조건문
HAVING - GROUP BY 로 집계했을때 사용하는 조건문
